1. 背景与目标

V1 客户端当前对未登录用户采用「橱窗 + 硬拦截」策略：UI 可看，按钮一点就硬跳登录页，没有任何前置提示。Activity / Wheel / Promotion 三个 Tab 内共有 30+ 个硬跳入口，导致访客踩雷、登录漏斗失血，邀请规则等本应公开的内容也被锁在登录后。

本期目标：用「软拦截抽屉」替代硬跳，让用户在被引导登录前能完整体验橱窗内容；同时把公共内容由 B 升 A，开放给未登录用户访问。

核心指标目标值
未登录用户在三个 Tab 的停留时长+15%
软拦截抽屉「立即登录参与」点击率≥ 25%
公共内容开放面活动咨询详情 / 邀请规则 / 返佣比例 由 B 升 A

6. 活动咨询 banner 分发规则

6.1 判定优先级

点击 banner 时按以下顺序判定：

  1. JumpLinkType = 1（子游戏）→ 未登录软拦截
  2. Visibility = 1（运营标记需登录）→ 未登录软拦截
  3. 否则按 JumpType 默认行为
核心原则：目标页的访问类决定行为，不是入口的访问类。同一活动详情页无论从哪个入口进入，未登录用户体验都保持一致。

6.2 JumpType 默认行为

JumpType名称默认行为
1富文本直接进富文本详情页（A 类）
2外链在 WebView 内打开外链 URL（建议先展示中转页提示用户即将离开）
3图片全屏图片预览
4内链按目标页 A/B 类判定，目标页 B 类则未登录软拦截
5客服沿用现有客服跳转逻辑，本期不改动

6.3 运营后台配置要求

  • 活动咨询配置表单新增「访问权限」单选项：
    • 公开（未登录可见，默认）
    • 需登录后访问
    对应字段 Visibility = 0 / 1
  • 选择 JumpType=4 内链且目标路由属于 B 类时，弹提示告知运营该入口未登录用户会被软拦截

6.4 存量数据迁移规则

本字段新增上线时，需对历史 banner 数据做一次性智能迁移，不允许全部默认设为「公开」，避免出现「列表显示公开但运行时实际被拦截」的语义不一致。

迁移规则（按以下顺序判定，命中即设值）：

条件迁移后 Visibility理由
JumpLinkType = 1（子游戏）1 需登录子游戏必须登录，与运行时规则①保持一致
JumpType = 4（内链）且目标路由属于B 类白名单1 需登录目标页本身需登录，运营后台展示与运行时行为一致
其他所有 banner（含 JumpType=1/2/3/5、JumpType=4 跳 A 类等）0 公开本就属于公开内容

B 类路由白名单由开发与运营共同确认，初版建议至少包含：

  • /wallet/*（钱包相关，含充值 / 提现 / 交易记录）
  • /my/*（我的中心相关）
  • /promotion/partner-rewards、/promotion/subordinate-data、/promotion/commission-detail 等 Promotion 内 B 类子页
  • /activity/* 中各活动详情页（如 First gift / Invitation bonus 等）

迁移脚本由开发实施，运营仅需提供 B 类路由清单。后续如发现遗漏可手工修正个别 banner 的访问权限。

7. 登录回跳逻辑

V1 已具备登录成功回跳能力，沿用现有机制。redirectTarget 由客户端本地缓存，有效期 30 分钟，超时跳首页。多次拦截以最后一次为准。

8. 横切要求

8.1 多语言

本期所有新增文案必须沿用 V1 现有多语言体系，覆盖站点支持的全部语言（中文 / 英文 / 等）。需翻译的文案集：

  • Activity 价值钩子 hero 三段（标题 / 副文 / CTA）
  • 软拦截抽屉的所有标题与描述（详见 §5.2 全表）
  • Wheel Tab 邀请转盘相关提示文案
  • 邀请规则 / 返佣比例富文本内容（如运营录入需要多语言版本）

翻译要求：

  • 同步上线：避免出现「中文已上但其他语言未上」的体验断层
  • 缺译 fallback：若某语言文案缺失，前端应 fallback 到英文，不允许直接显示 key（如 intercept.title）
  • 文案集中管理：建议所有抽屉文案归入同一 i18n namespace（如 intercept），便于运营 / 翻译统一审校

8.2 异常与降级

异常场景处理方式
A 类接口失败（如邀请规则）前端展示「内容加载失败，请稍后重试」，不要硬跳登录
IsOpenInvitedWheel 配置缺失 / 异常默认按「活动未开启」处理（更安全，避免误展示橱窗）
banner 配置异常（缺 JumpType 字段）点击无响应或提示「该活动暂不可用」
redirectTarget 超过 30 分钟直接丢弃，登录成功后跳首页，不报错
登录页不存在 / 跳转失败降级到原硬跳逻辑

9. 名词与易误解数据速查

9.1 名词约定

术语含义
A 类（开放）未登录可正常访问，无任何拦截
B 类（需登录）未登录点击触发软拦截抽屉
软拦截底部抽屉，不切换路由，提供「立即登录参与 / 暂不登录」两个动作
硬拦截直接跳登录页（本期 Home / Mine 仍保留）
JumpTypebanner 跳转类型（1 富文本 / 2 外链 / 3 图片 / 4 内链 / 5 客服）
Visibilitybanner 访问权限（0 公开 / 1 需登录）
JumpLinkType内链子类型，=1 为子游戏（必须登录）
IsOpenInvitedWheel后台已有的邀请转盘活动开关，Wheel Tab 入口 A/B 判定唯一依据
