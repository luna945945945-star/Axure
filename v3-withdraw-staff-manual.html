▲ 收起

组长 1 · 你是谁，你管什么 — 组长角色定位 + 3 大核心职责

⭐
组长不需要点【开始派单】自己接单。你的工作是监控 + 调控 + 处理组员转派的异常单。
你的 3 大核心职责
① 监控本组员工状态：通过员工实时状态看每个组员的单量、暂停、超时情况
② 调控员工接单：积压员工 → 点【停止派单】或把订单转派给空闲员工；空闲员工 → 让他【开始派单】或转派单子给他
③ 处理组员转派给你的异常单：组员遇到大单 / 风控 / 资料疑问会转派给你，你处理
📌
记住：组员"异常暂停"后只能由你恢复——他们自己点不动浮窗。
每天开班务必先扫一遍"员工实时状态 → 上班(暂停超时异常) / 上班(未审核超时异常)" 这两个 Tab。

多 3 · 商户切换的使用场景 — 何时需要切商户、何时不需要

💡
再次提醒：切换商户不会改变系统给你派单的范围。
· 派单范围 = 你的"授权出款商户"配置（主管建用户时勾的），恒定，不会因为切换顶部商户而变
· 切换顶部商户 = 改变"其他页面"看的数据（如提现记录、报表）
· 操作台是全局的，切换商户对它没有影响
⚠️
切换商户会触发全页刷新。
详情弹窗已选的通道、已填的备注、已勾选的批量订单 —— 全部丢失。
切换前必做：① 关闭所有打开的详情弹窗 ② 确认手上无未提交的操作。

出款员工操作台详解

主战场 · 所有职位都用的核心页面 · 8 子节

📍 路径：财务管理 → 提现订单管理 → 出款员工操作台

操作示例 + 说明

标区域 / 按钮说明
1出款员工操作台 Tab当前页签，与"出款看板 / 当前提现订单 / 全部提现记录" 等平级
2个人状态显示当前状态徽章（如 上班-派单中），与右下角浮窗联动
3顶部统计卡片今日已工作时长 · 今日暂停次数 · 今日暂停总时长 + 【手动刷新】【提醒/规则】
4Pills 过滤总单数 X · 待审核 X · 审核中 X — 点击切换列表
5行操作 【开始审核】待审核状态显示；点击开始审核计时
6行操作 【指派组长】遇到大单 / 风控 / 资料疑问 → 转给组长处理
7右下角浮窗（派单中）状态切换主入口：上班 / 开始派单 / 暂停 / 下班（可拖拽，位置自动记忆）
⭐
关键认知：操作台是全局页面，不受顶部切换商户影响
· 顶部切换商户 → 其他页面会换数据（如提现记录、报表等）
· 但操作台始终展示你所有授权出款商户的派单，不会因为顶部切换而变化
· 派单依据 = 你的"出款权限配置 → 授权可出款的商户"（主管建用户时勾的），不是基础配置的"归属商户"

两个"商户"概念别搞混

概念在哪儿配控制什么
归属商户（基础配置）系统用户管理 → 公用配置 → 归属商户你能登录访问哪些商户的数据（顶部下拉的范围）
授权出款商户（出款权限）系统用户管理 → 编辑 → 出款权限 → 授权可出款的商户系统派哪些商户的提现订单给你审核（出款列表的范围）
💡
授权出款商户 ⊆ 归属商户（主管建用户时只能在归属范围内勾出款商户）

1.1 顶部员工状态 — 当前工作状态徽章的含义和恢复方式

状态标签含义谁能恢复
未上班今天还没打卡 / 已下班自己浮窗【上班】
上班(未开启接单)已打卡但未点【开始派单】自己浮窗【开始派单】
上班(接单中)正常接单中—
上班(主动暂停中)自己主动暂停（手上的单会回流到订单池）自己浮窗【恢复…】
上班(管理设置暂停接单)主管 / 组长点了【停止派单】（手上单不影响）主管 / 组长【开始派单】
上班(暂停超时异常)主动暂停超过组别配置时长主管 / 组长【解除异常】
上班(未审核超时异常)订单长时间未审核被系统标异常主管 / 组长【解除异常】
禁用账号级 — 出款权限被关闭找主管开权限

1.2 右下角浮窗（状态切换主入口）⭐ — 上下班 / 暂停 / 恢复的快捷操作入口

🖱️
浮窗可拖拽，位置自动记忆（打卡上班、暂停、恢复、下班等状态的点击切换）。
当前状态主按钮（大）次级按钮
未上班【上班】—
上班(未开启接单)【开始派单】【下班】
上班(接单中)【暂停】【下班】（二次确认）
上班(主动暂停中)【恢复并派单】【恢复但不派单】+【下班】
⚠️
· 【暂停】= 手上的单会回流到订单池（适合临时离开）；不是"只停接新单"
· 想"只停接新单、继续审手上的" → 让主管 / 组长 给你【停止派单】，不是自己【暂停】
· 暂停不需要填原因，直接点按钮即可
· 【下班】在接单中状态会弹二次确认（防误操作）
· 异常状态下浮窗按钮变灰，需要主管 / 组长【解除异常】

【确认取消】（拒绝）
适用：会员账户异常 / 风控提示 / 银行卡异常 / 重复提现
步骤：弹窗底部点按钮 → 二次确认（必填取消原因）→ 订单状态变 拒绝

FAQ + 状态速查 + 入口速记

8 条 FAQ · 状态枚举 · 查询入口

3.1 常见问题（8 条） — 日常审单中最容易遇到的问题汇总

Q1. 我点了【提交三方】后订单卡在"已提交"超过 30 分钟还没回调，怎么办？
A：正常三方回调是 5-30 分钟。超 30 分钟未回调：
① 先看通道详情的日志报错信息
② 看是不是通道本身有问题（去看其他订单同通道是否也卡）
③ 上报 组长 / 技术，可能需要查接口或人工对账
不要随便用【人工确认】，除非确认线下钱已到。
Q2. 我的金额区间是 500~10万，但组长转派给我一笔 20 万的单，能审吗？
A：能。"指派订单配置"只控制系统自动派给你的单的金额范围；组长 手动转派 给你的单不受此限制，正常审核即可。
Q3. 我被系统标记为"异常暂停"了，浮窗按钮变灰，自己怎么恢复？
A：你自己恢复不了。需要找组长（或更高管理员）在「员工实时状态」页面找到你的账号 → 点【解除异常】。
恢复后你的浮窗按钮才能再次点击。
Q4. 派单中状态，我点【暂停】后正在审核的订单会怎样？
A：员工自己点【暂停】 = 手上的订单会回流到订单池，派给下一位员工。
这是你临时离开（如午休、临时有事）的场景——把单交出去让别人接着审。

"只停接新单、继续审手上的"是组长的【停止派单】功能，不是你的【暂停】：
· 你的【暂停】 → 手上的单回流 + 不接新单
· 组长【停止派单】（给你）→ 手上的单不受影响 + 不接新单，需要组长再点【开始派单】才恢复

注意：暂停时长超过组别配置（默认 5 分钟）会变成"异常暂停"，那时需要组长【解除异常】。
Q5. 我的"审核时长"列变红了，会被扣绩效吗？
A：变红 = 超过了组别配置的"已审核超时"。
① 订单会被系统自动回流给下一员工，不再属于你
② 你的"超时次数"指标会 +1（在员工报表里能看到）
③ 是否影响绩效看公司具体考核制度，建议控制审核时长在 5 分钟内
Q6. 我指派给组长的订单还能找回吗？
A：不能自己找回。订单指派后已经从你的列表消失，需要组长再转回来给你。
建议指派时填清楚指派备注（为什么转、怎么处理），避免来回拉扯。
Q7. 详情弹窗里的"会员信息""游戏记录"Tab 我看不到，是为什么？
A：这几个 Tab 是按权限动态显示的。看不到说明你的角色没有对应菜单权限（如会员管理、游戏管理）。
如果业务上需要看会员信息来判断，找主管开通对应权限。
Q8. 切换商户时正在审核某单，弹窗突然关了订单丢了吗？
A：切商户会触发全页刷新，弹窗关闭，你在弹窗里没保存的备注 / 选通道操作会全部丢失。订单本身不会丢，但要重新打开重新选。
建议：审核完单 → 再切商户。
