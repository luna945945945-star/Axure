📌 业务规则

规则说明
判定时间窗从用户上次提现完成时间（无上次则取注册时间），到本次提现申请时间之间
判定字段银行名称 / 收款人姓名 / 银行账号 / IFSC（若有）。任一字段在时间窗内被新增或修改，即触发提示
对应数据源tab_WithdrawalBankCard 的 AddTime / UpdateTime / DeletedTime；与本次订单的银行卡 ID 关联
展示位置① 列表「银行卡详细」列底部；② 审核订单详情弹窗「银行卡详情」区域底部
展示形式默认收起，仅一行红字"⚠️ 用户提现银行信息变更，请核查 ▼"；点击展开显示最近修改日期（不展示修改内容详情）
为何不展示修改内容由审核人员自行核实具体变更内容，避免界面冗余；展开后只给一个时间锚点
显示条件仅待审核订单显示（已通过/已拒绝订单不显示，减少干扰）
是否拦截审核不拦截，仅展示提示，由运营自行判断是否通过
