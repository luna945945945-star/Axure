JumpType 取值与判定
1·富文本公开 后台录入的活动公告 / 规则文案，直接渲染富文本详情
2·外链公开 跳第三方 URL（合作伙伴 / 官方网站），WebView 直接打开
3·图片公开 海报 / 宣传图，全屏预览，不涉及账户数据
4·内链公开需登录 跳站内页面，按目标页是否依赖账户数据判定
4·子游戏需登录 JumpLinkType=1，需要账户身份才能进入游戏
5·客服公开 支持未登录工单（访客咨询），登录用户进入个人专属工单
⚖ 判定优先级
① JumpLinkType==1（子游戏）→ 需登录
② 后台 Visibility 字段：
  0 = 公开，1 = 需登录
③ 都未命中按 JumpType 默认：
  1 / 2 / 3 / 5 = 公开
  （5·客服未登录走访客工单）
